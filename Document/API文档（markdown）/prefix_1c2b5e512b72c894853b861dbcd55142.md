
    
**简要描述：** 

- 删除商品接口

**请求URL：** 
- ` http://192.168.2.201/commodity/deletecom/ `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |是  |int |要删除的商品id   |


 **返回示例**

``` 
  {
    error_code: "0"
  }
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|error_code |int   |执行完成 |

 **备注** 

- 若非0则说明出现错误

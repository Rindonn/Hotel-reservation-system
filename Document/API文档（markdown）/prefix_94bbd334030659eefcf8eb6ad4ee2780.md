
    
**简要描述：** 

- 用户登出接口

**请求URL：** 
- ` http://192.168.2.201/login/logout/ `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|token |是  |string |用户令牌   |

 **返回示例**

``` 
  {
    "error_code": 0,
	"msg": "Logout success"
  }
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|error_code |int   |错误代码,0为正常执行   |
|msg | String | 返回信息|

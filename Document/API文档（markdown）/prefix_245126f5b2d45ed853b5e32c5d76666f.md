
    
**简要描述：** 

- 检查权限接口

**请求URL：** 
- ` http://192.168.2.201/login/check `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|token|是|varchar|用户token

 **返回示例**

``` 
  {
    "error_code": "0",
    "status": 2
  }
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|error_code |int   |正常执行  |
|status|int|用户权限|

 **备注** 

- 权限0为被ban/删除，权限为1是正常用户，权限2是管理员


    
**简要描述：** 

- 删除用户接口

**请求URL：** 
- ` http://192.168.2.201/user/deleteuser/ `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|userid |是  |int |用户id   |

 **返回示例**

``` 
  {
    "error_code": 0
  }
```
 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|error_code |int   |正常执行|

 **备注** 

- 若非0则说明出现错误


    
**简要描述：** 

- 添加商品接口

**请求URL：** 
- ` http://192.168.2.201/commodity/insert/ `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|name |是  |string |商品名称   |
|kind |是  |string | 商品种类    |
|stock     |是  |int | 商品库存   |
|disc     |是  |String | 商品描述   |
|img     |是  |String | 商品图片地址   |
|price     |是  |decimal | 商品价格   |


 **返回示例**

``` 
  {
    error_code: "0"
  }
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|error_code |int   |执行完成  |

 **备注** 

- 若非0则说明出现错误
